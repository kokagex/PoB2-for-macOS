# 村の天気予報・掲示板ツール

天気を知らせ、神への祈り（メッセージ）を記録する、村人と神を繋ぐツールです。

## 概要

このツールは、村人が毎日の天気予報を確認し、神への思いを掲示板に投稿できるコマンドラインアプリケーションです。

- 架空の天気を提供する天気予報機能
- 村人の声を記録する掲示板機能
- シンプルで使いやすいメニュー形式

---

## 機能一覧

### 1. 天気予報機能

指定した都市名を入力すると、架空の天気と気温を表示します。

**特徴:**
- 天気パターン: 晴れ、曇り、雨、雪、嵐など（ランダム生成）
- 気温範囲: -10℃〜35℃
- 毎回異なる天気が表示されます

### 2. 掲示板機能（神への一言）

村人のメッセージを入力し、ファイルに保存します。

**特徴:**
- メッセージの保存: `data/messages.json`
- 記録情報: 日時、投稿者名、メッセージ本文
- 過去メッセージの一覧表示機能
- すべての投稿が永続的に保存される

## 使用方法

### インストール

**必要環境:**
- Python 3.x
- pip（Python パッケージマネージャー）

**セットアップ:**

```bash
# プロジェクトディレクトリに移動
cd village_tool

# 必要な依存パッケージをインストール（もしあれば）
pip install -r requirements.txt
```

### 実行方法

```bash
python main.py
```

メニューが表示されます。数字を入力して機能を選択します。

### コマンド例

#### 天気予報を確認する

```
村の天気予報・掲示板ツール
============================
1. 天気予報を見る
2. 掲示板にメッセージを投稿
3. 過去のメッセージを見る
4. 終了

選択してください (1-4): 1
都市名を入力してください: 東京
```

**出力例:**
```
東京の天気: 晴れ
気温: 15℃
```

#### 掲示板にメッセージを投稿

```
選択してください (1-4): 2
投稿者名を入力してください: 太郎
メッセージを入力してください: 今日も良い一日でありますように
```

**確認メッセージ:**
```
メッセージを保存しました
投稿日時: 2026-01-28 10:30:45
投稿者: 太郎
```

#### 過去のメッセージを確認

```
選択してください (1-4): 3
```

**出力例:**
```
=== 掲示板の全メッセージ ===

1. 太郎 (2026-01-28 10:30:45)
   今日も良い一日でありますように

2. 花子 (2026-01-28 11:15:30)
   神様、加護をお願いします

合計 2 件のメッセージ
```

## ファイル構成

```
village_tool/
├── main.py           # エントリーポイント（メニュー制御）
├── weather.py        # 天気予報機能モジュール
├── bulletin.py       # 掲示板機能モジュール
├── data/
│   └── messages.json # メッセージ保存ファイル（JSON形式）
└── README.md         # このファイル
```

### 各ファイルの役割

**main.py**
- ユーザーとのインタラクション（メニュー表示、入力受け付け）
- 各機能モジュールの呼び出し制御
- プログラムのメインループ

**weather.py**
- 架空の天気データ生成
- 気温のランダム生成
- 天気情報の表示

**bulletin.py**
- メッセージのファイル保存処理
- JSONファイル操作
- メッセージの読み込みと表示

**data/messages.json**
- 投稿されたすべてのメッセージを保存
- JSON形式で日時、投稿者名、内容を記録

## トラブルシューティング

### 「messages.json が見つかりません」というエラー

`data/` ディレクトリが存在することを確認してください。ない場合は手動で作成します。

```bash
mkdir data
```

### Python コマンドが認識されない

Python がインストールされているか確認してください。

```bash
python --version
```

インストールされていない場合は、[Python公式サイト](https://www.python.org/)からダウンロードしてください。

## 注意事項

- `data/messages.json` は重要なデータファイルです。削除しないでください
- メッセージは一度投稿されると、手動で削除する必要があります
- 架空の天気であることにご注意ください（実際の天気予報ではありません）
