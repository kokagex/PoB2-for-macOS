**はじめに**
1. Macでの準備（最初の一回だけ）
    VS Codeをインストール: 公式サイトからMac版をダウンロードしてインストールします。
    Gitをインストール: Macのターミナル（Command + スペースで「terminal」と検索）を開き、git --version と打ってみてください。もし入っていなければ「インストールしますか？」とポップアップが出るので、そのまま進めるだけでOKです。
    日本語化: Windowsと同じように、拡張機能から「Japanese Language Pack」を入れればメニューが日本語になります。

2. Macで作業を再開する手順
    新しいMacで作業を始めるのは、Windowsのときよりずっと簡単です。

    手順A：VS Codeでスマートに再開する（おすすめ）
        VS Codeを開き、[リポジトリをクローン]（または [ソース管理] タブ ＞ [リポジトリをクローン]）をクリック。
        [GitHub からクローン] を選択。
        GitHubにログインすると、あなたのリポジトリ claudecode01 が一覧に出るので選択します。
        保存場所を決めたら完了！

    手順B：コマンドで再開する
        ターミナルを開いて、以下の1行を打つだけです。
        Bash 
        git clone https://github.com/kokagex/claudecode01.git

**引継ぎ**
1. GitHubからコードをダウンロード（クローン）する
    Macの「ターミナル」（Terminal.app）を開いて、プロジェクトを保存したい場所（デスクトップなど）に移動してから、リポジトリをコピーします。

    Bash 

    1-1. デスクトップに移動する場合
        cd ~/Desktop

    1-2. GitHubからプロジェクトをダウンロード
        git clone https://github.com/kokagex/claudecode01.git

    1-3. 作成されたフォルダの中に入る
        cd claudecode01

2. VS Codeでフォルダを開く
    ターミナルでそのまま以下のコマンドを打つと、VS Codeでそのプロジェクトが開きます。

    Bash 
    code .

    （もし code コマンドが効かない場合は、VS Codeを普通に開いて「ファイル」→「開く」から claudecode01 フォルダを選んでください）

3. Claude Codeを起動してログインする
    プロジェクトのフォルダ内で Claude Code を起動します。

    Bash 
    claude

    ログイン: 初めてMacで使う場合、ブラウザが立ち上がり認証を求められるので承認してください。

    作業の継続: 起動したら、Claudeに「Windowsでやっていた作業の続きをしたい」と伝えてみてください。

4. もし「環境」に違いがある場合（注意点）
    WindowsとMacでは、実行環境（Node.jsのバージョンやパスなど）が微妙に異なることがあります。もし依存パッケージ（node_modules）が必要なプロジェクトなら、起動前に以下を実行しておくと確実です。

    Bash 
    npm install

    まとめ：引き継ぎのコツ
        Claude Codeは、そのフォルダ内にある CLAUDE.md や Gitの履歴を読み取って「何をやっていたか」を把握します。
        Claudeへの指示例: 「GitHubからこのプロジェクトをクローンしてきたところです。前回の作業内容を確認して、続きのタスクを教えて」
        このように話しかければ、Claudeが現在のコードの状態をスキャンして、スムーズに作業を再開してくれます。
        Macでの起動はうまくいきましたか？もし「コマンドが見つからない」などのエラーが出たらすぐに教えてくださいね！
